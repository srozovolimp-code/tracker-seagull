# Tracker Seagull

Функциональный трекер задач в стиле рабочего пространства Teamly.

Возможности:
- таблица задач с группировкой по командам;
- канбан, календарь, диаграмма Ганта и аналитика;
- создание, редактирование и удаление задач;
- фильтры, поиск, импорт и экспорт JSON;
- локальное сохранение данных в браузере.

Стек: Next.js, React, TypeScript, Tailwind CSS.
